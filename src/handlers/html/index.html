Blossom Server

This is a Blossom protocol server that allows you to store and retrieve blobs of data.

To use this server, you can send HTTP requests to the following endpoints:

  • GET /<hash> - Retrieve a blob by its hash
  • HEAD /<hash> - Check if blob exists
  • DELETE /<hash> - Delete blob
  • PUT /<upload> - Upload a new blob
  • GET /list/<pubkey> - List all blobs associated with a public key
  • PUT /mirror - Copy blobs from another Blossom server

For more information on how to use the Blossom protocol, please see the Blossom documentation.

Client Software

If you're looking for a more user-friendly way to interact with this Blossom server, you can try using Blossom Drive. Blossom Drive is a graphical client that allows you to easily upload, download, and manage blobs on this server.

By using Blossom Drive, you can take advantage of the Blossom protocol's features without having to manually send HTTP requests. Simply click the link to go to Blossom Drive (hosted by Hzrd149), configure it to connect to this server, and you're ready to go!
